# Screenshot Placeholder

Для создания реального скриншота:

1. Запустите демо: `npm run dev`
2. Откройте в браузере
3. Сделайте скриншот компонента
4. Сохраните как `docs/screenshot.png`
5. Обновите README с реальным путем

Пример кода для README:
```markdown
## Preview

![Cookie Consent Drawer](docs/screenshot.png)
```
